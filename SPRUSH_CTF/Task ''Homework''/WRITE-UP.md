1. При заходе на сервер видим, что нас просят решить много примеров по математике
   При правильном решении запросы продолжаются, при неверном - обрываются

2. Для автоматизации используем Python-script с сокетами, получая в строчном формате пример
3. Обрабатываем его, убирая пробелы, знак равенства и переводя в строку из bytes
4. С помощью либы numexpr решаем пример из строки и результат в закодированном виде отправляем на сервер
5. Выводим каждое полученное сообщение и при выводе флага программа крашнется, предварительно выведя флаг
